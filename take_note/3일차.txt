2.연산자
	1)산술 연산자
		+ : 더하기
		- : 뺄셈
		* : 곱셈
		/ : 나눗셈
		   정수/정수 => 정수
		   실수/정수 => 실수
		   정수/실수 => 실수
		% : 나머지
		   정수 % 정수 만 가능
	 	   짝수 : 숫자 % 2 == 0
		   홀수 : 숫자 % 2 == 1
		   n의배수 : 숫자 % n == 0

	2)증강연산자(++,--)
		-전위형 : 연산(또는 실행) 전 증강 수행(++변수명, --변수명)
			나(변수) 먼저 증가/감소하고 계산식에 참여
			실행 순서가 1위
		-후휘형 : 연산(또는 실행) 후 증강 수행 (변수명++,변수명--)
			계산 먼저하고 나(변수)를 증가/감소 함
			실행 순서가 마지막
	
	3)비교 연산자 : 연산 결과가 true 또는 false로 나옴
		> : 크다,초과
		>= : 이상, 크거나 같다.
		<: 작다, 미만
		<= : 작거나 같다, 이하
		== : 같다
		!= : 같지 않다

		==,1= : 스택메모리의 값을 비교 (기본 자료형일 때는 값을 비교, 참고 자료형 일 때는 번지 비교)

	4)논리 연산자(&&, ||, !, ^)
		-결과는 true 또는 false 로 표시
		-논리 연산자의 좌우에는 반드시 true/false 만 올 수 있음
		-비트 연산자나 논리 연산자의 연산은 같다. 그러나
		-비트 연산자보다는 논리연산자가 속도가 빠르다. (Short Circuit)
			비트 연산자는 Short Circuit 적용 되지 않음
		=short circuit : 연산 수행시 결과가 이미 확정된 경우 나머지 연산과정을 수행하지 않는 것




		값1 &&	값2	결과
		true	true	true	
		true	false	false	
		false	true	false	
		false	false	false	
				

		값1  ||	값2	결과
		true	true	true	
		true	false	true
		false	true	true	
	 	false	false	false

	
		값1  ^	값2	결과(두 조건이 다르면 true)
		true	true	false	
		true	false	true	
		false	true	true	
		false	false	false

		!값1
		! true	false
		! false	true
		





		국,영,수 모두 80점 이상이면
			=>국>=80&& 영>==80&& 수=>80
			=> && : 그리고,and, 모두 ~~
		국,영,수 중 적어도 한과목이 80점 이상(세과목 중 하나이상만 80점 이상)
			=>국>=80 ||  수>=80 || 영>=80
			=> || : 또는 ,or
			=>세 과목 중 하나 이상만 80점 이상이면 나머지는 어떤 값이든 상관 없음

	5) 대입연산자/복합대입연산자
		= : 오른쪽의 내용을 왼쪽에 대입
		+= : a = a + 1 => a += 1, a=a +7 => a+7
		-= : a = a - 1 => a -= 1, a = a - 7 => a - = 7
		*= : a = a*1 => a *=1, a= a *7 => a*=7
		/=
		%/
		...

	6) 연산자 우선 순위
		-괄호와 대괄호가 연산순위가 제일 높음
		-산술 > 비교 > 논리 > 대입
		-단항 > 이항 > 삼항
		-연산자의 연산 진행 방향은 왼쪽에서 오른쪽으로
		-단, 단항,대입 연산자만 오른쪽에서 왼쪽으로
